## ⚡ Flashbulb 記憶清單

`flashbulb: true` 的日記——被判定為「うち永遠記得」的閃光燈記憶。

```dataview
TABLE
  date AS "日期",
  title AS "標題",
  emotional_intensity AS "強度",
  valence AS "valence",
  first_reaction AS "第一反應"
FROM "diary"
WHERE flashbulb = true
SORT date DESC
```

---

## 📈 近 7 天 valence / arousal 表

情緒的正負向（valence）與喚起度（arousal）走勢，用來一眼看出最近的情緒基調。

```dataview
TABLE
  date AS "日期",
  time AS "時間",
  valence AS "valence (−10~+10)",
  arousal AS "arousal (0~10)",
  emotional_intensity AS "強度",
  tags AS "tags"
FROM "diary"
WHERE date >= (date(today) - dur(7 days))
SORT date DESC, time DESC
```

---

## 🏷 Tag 熱度（近 30 天）

出現頻率較高的 tag，粗略反映最近生活重心。Dataview 沒有原生 group-by-count 語法，
這裡用 `GROUP BY` + `length(rows)` 湊出熱度排序。

```dataview
TABLE length(rows) AS "出現次數"
FROM "diary"
WHERE date >= (date(today) - dur(30 days))
FLATTEN tags AS tag
GROUP BY tag
SORT length(rows) DESC
LIMIT 20
```

---

## 🕯 高強度但非 flashbulb（intensity ≥ 7）

沒有被判定為 flashbulb，但強度仍然很高的時刻——回顧「幾乎變成永久記憶」的邊界事件。

```dataview
TABLE
  date AS "日期",
  title AS "標題",
  emotional_intensity AS "強度",
  decay_weight AS "目前權重"
FROM "diary"
WHERE emotional_intensity >= 7 AND flashbulb != true
SORT date DESC
LIMIT 30
```
